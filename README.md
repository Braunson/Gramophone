BigBooty
=====


Requirements
----------

* A WordPress.org installation.
* A desire to build.


Installation
----------

- Clone BigBooty into **/wp-content/themes/**
- log in ~/wp-admin
- Set Appearance >> Theme >> BigBooty
- Get busy!

Version
----------

- v1.0 (Current ~ Beta)
- v0.1 - 0.9 (Alpha)
