BigBooty
=====


Requirements
----------

* A WordPress.org installation.
* A desire to build.


Installation
----------

- Clone the BigBooty Repo
- Either upload via Theme Options
- 
*or*

- Place in **/wp-content/themes/**
- Get busy!

Version
----------

**v0.1 - 0.9 (Alpha)**

BigBooty is *not fit* for the wild
